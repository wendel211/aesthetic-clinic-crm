# CRM Estetica Pro

CRM para clinicas de estetica, studios de beleza e profissionais que vendem procedimentos recorrentes e pacotes.

## Stack atual

- Next.js 16 com App Router
- TypeScript
- Tailwind CSS 4
- ESLint

## Escopo atual

A primeira entrega tecnica ja inclui uma tela operacional em portugues do Brasil com foco em:

- dashboard do dia
- agenda de atendimentos
- clientes em aquecimento
- saude dos pacotes
- retornos recomendados
- fila de WhatsApp

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`
